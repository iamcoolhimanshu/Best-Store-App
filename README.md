# BestStore - Product Management System

A simple CRUD (Create, Read, Update, Delete) web application built using **Java**, **Spring Boot**, **Thymeleaf**, and **MySQL**.

This project provides a straightforward web interface to manage a list of products in a database.

---

## üöÄ Core Features

This application demonstrates the four basic functions of persistent storage:

* **Create:** Add new products (e.g., name, brand, price) to the database using a web form.
* **Read:** View a complete list of all products displayed in a table.
* **Update:** Select a product from the list to edit its details.
* **Delete:** Remove a product from the database.
